# Coursera_CleaningData_Proj
This repo contains the work done in the course project in fulfilment of the Cleaning Data course in Data Science Specialization by John Hopkins University through Coursera.

The code has ben created and tested in R-studio Version 0.98.1091 – © 2009-2014.

Steps to run
------------

1. Set working directory in R-studio to \UCI HAR Dataset
2. Copy run_Analysis.R inot working directory
3. Type source("run_Analysis.R") at the command prompt and press enter.
4. The output file tidydata.txt will be output when the script completes.

The file CodeBook.txt contains a brief description of the variables captured in tidydata.txt
